**** Cách tạo file config cho phần mềm
>>1.Tạo file ds.cfg
>>2.Dòng 1: tên hệ điều hành (Vd: Windows -> win; Linux -> linux)
>>3.Dòng 2: N học sinh và M bài thi
>>4.N dòng tiếp theo: Tên học sinh (tên thư  mục trong BaiDuThi) và ngôn ngữ lập trình (Vd: C++ -> cpp, C -> c)
>>5.M dòng tiếp theo: Tên bài thi, số test và thời gian chạy
**** Lưu ý!:
>> Hiện tại phần mềm vẫn đang trong giai đoạn phát triển nên có thể có những lỗi phát sinh trong quá trình sử dụng
>>và mới chỉ hỗ trợ duy nhất 2 ngôn ngữ C++ và C. Bù lại, phần mềm này là phần mềm mã nguồn mở nên người dùng có thể
